A tree is a type of list that can be organized into folders of which can be expanded and collasped as a way to filter (or chuck) the content for the user to absorb and understand easier. Typically trees tend to also include methods for for searching, text based filtering, ability to make group. Each line of data may also include a way to toggle its visibility.
